v0.1 Released

Oct 28th, 2012

Dentro 0.1 was released.

Release notes:

  • File I/O ++ Reads (basic) OPML files (standard not fully implemented) ++ Writes to OPML
  • Tree manipulation ++ Enter creates new, empty item under selected item, and focus on it ++ Ctrl+delete deletes current item and all its children nodes +++ Just delete only functions on node text ++ Tab indents in current node immediate (previous) sibling ++ Shift+Tab indents out current node ++ Alt+Enter or click on the node’s bullet toggles expand/collapse node +++ reveals all sub-nodes under current node +++ close also closes all sub-nodes
